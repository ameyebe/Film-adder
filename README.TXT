PROJECT TITLE: FIlm recommender
PURPOSE OF PROJECT: Too allow the user to add movies to a hashmap, and choose them based on genres
VERSION or DATE: 1
HOW TO START THIS PROJECT: Right click film GUI class and select main routine.
AUTHORS:Benjamin Ameye
USER INSTRUCTIONS: This code will allow you to add movies to a list, as well as include their director, score (0 to 100)
and genre. Then you'll be able to print them based on what you like. Select one of the buttons on the left hand side to 
add a film, print based on genre, or print based on the director. THere will also be a slider you can use to adjuct the films score.
Selecting the score button will print movies with a score equal to or greater than the score the slider is set to. Note, you can't add
movies that are already in the hash map.
